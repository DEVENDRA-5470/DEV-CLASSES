16. Kaun Banega Crorepati (KBC) Program in JavaScript
Objective:
Create a simple quiz program based on the Kaun Banega Crorepati (KBC) format using JavaScript. The program should ask five questions and check the user's answers using if-else statements.

Instructions:

The program should prompt the user with five questions, each with four options (A, B, C, D).
For each question, the program should provide feedback on whether the answer is correct or incorrect.
At the end of the quiz, the program should display the total score out of five questions.
Questions:

What is the capital of France?
A. Berlin
B. Madrid
C. Paris
D. Rome

Which planet is known as the Red Planet?
A. Earth
B. Mars
C. Jupiter
D. Venus

What is the largest mammal in the world?
A. Elephant
B. Blue Whale
C. Giraffe
D. Great White Shark

Which element has the chemical symbol 'O'?
A. Oxygen
B. Gold
C. Silver
D. Helium

What is the hardest natural substance on Earth?
A. Gold
B. Diamond
C. Iron
D. Quartz

Implementation Steps:

Use prompt() to collect answers from the user for each question.
Use if-else statements to check the user's answer against the correct answer.
Maintain a score counter that increments for each correct answer.
Use alert() to provide feedback for each question and display the final score after all questions are answered.
