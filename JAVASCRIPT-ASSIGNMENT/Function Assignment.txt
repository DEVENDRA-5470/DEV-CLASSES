### Assignment Functions (JavaScript)

 Problem1:
Write a function to count the total odd and total even numbers by taking the start point and end point as arguments.

 Problem2:
Write a function to filter out the greatest string from a given array `['python', 'javascript', 'arguments', 'java']`.

 Problem3:
Write a function to filter out the smallest string from a given array `['python', 'javascript', 'arguments', 'java']`.

 Problem4:
Write a function to find all possible string data including characters given as an argument by input, from the array `['find', 'all', 'possible', 'string', 'data', 'including', 'character']`.

 Problem5:
Write a function to find the duplicate strings from a given array and an array provided by the user as arguments `['apple', 'mango', 'apple', 'pineapple', 'carrot', 'mango', 'banana']`.

 Problem6:
Write a function to count all characters from the string `'find all possible string data including character'` and a string passed by the user as arguments.

 Problem7:
Write a function to find all odd and even numbers by taking the start point and end point as arguments using array comprehension.

 Problem8:
Write a function to filter out all vowels and consonants from a string given by the user using the filter function.

 Problem9:
Write a function to reverse the array `[2, 3, 4, 5, 6, 7, 9, 23, 90]` and an array passed as arguments.

 Problem10:
Write a function to convert the array `[1, 2, 3, 4, 4, 5, 6, 7, 8, 8]` to `['a', 'a', 'a', 'a', 'a', 'a', 'a', 'a', 'a', 'a']`.

 Problem11:
Given the array: `[[1, 2, 3, 4, 5], ["python", 'java', 'javascript', 'mango'], [2, 4, 6, 8, 10], [30, 33, 54, 23, 34, 67, 89]]`
Output array:
```
[1, 2, 3, 4, 5]
["python", 'java', 'javascript', 'mango']
[2, 4, 6, 8, 10]
[30, 33, 54, 23, 34, 67, 89]
```
